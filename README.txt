How to Play

1. Open two Visual Studio Code windows

2. Open the ai_server folder

  2-1. Run main.py

3. Open the bowling_capt folder

  3-1. Run bowling_capt/lib/main.py

If you have an emulator(SDK35), launch it by pressing Ctrl + Shift + P, then run step 3 again.
